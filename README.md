📌 Task Manager Kanban

Este projeto é um organizador de tarefas estilo Kanban, desenvolvido com o objetivo de praticar e consolidar conhecimentos adquiridos durante meus estudos em desenvolvimento web.

A aplicação permite gerenciar tarefas de forma simples e visual, organizando-as em colunas como To Do, In Progress e Done, facilitando o acompanhamento do fluxo de trabalho.

🚀 Tecnologias utilizadas
Frontend
React
TypeScript
Tailwind CSS
Radix UI (componentes acessíveis e estilizados)
Zod (validação de dados)
Backend
Python (API)
Flask
MariaDB / MySQL
⚙️ Funcionalidades
✅ Criar novas tarefas
✏️ Atualizar status das tarefas
❌ Deletar tarefas
📋 Organização em formato Kanban
🔄 Integração com API backend
✔️ Validação de dados com Zod
🧠 Objetivo do projeto

Este projeto foi desenvolvido com foco em:

Praticar conceitos de frontend moderno com React
Trabalhar com gerenciamento de estado
Aprender integração com APIs REST
Utilizar validação de dados
Criar uma aplicação fullstack com Python + banco de dados
