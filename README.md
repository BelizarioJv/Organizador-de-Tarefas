📋 Gerenciador de Tarefas

Aplicação full-stack para gerenciamento de tarefas, desenvolvida com React + TypeScript no frontend e Node.js no backend. O sistema permite criar e gerenciar tarefas de forma simples e eficiente, com persistência de dados utilizando MariaDB.

🚀 Funcionalidades
✅ Criar novas tarefas
📄 Listar tarefas
❌ Excluir tarefas
🔜 Editar descrição da tarefa (em desenvolvimento)
🌐 Integração com API REST
🧪 Testes de API utilizando Postman
